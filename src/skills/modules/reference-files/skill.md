# Reference Files

讀取、列出與整理使用者提供的參考檔案，讓 Golem 回答時可以依據附件、文件或資料檔內容，而不是只靠記憶猜測。

## 使用時機

- 使用者提到附件、參考檔案、上傳檔案或「剛剛那份文件」。
- 使用者要求根據文件內容摘要、比對、改寫或抽取重點。
- 使用者需要確認目前可用的參考檔案清單。

## Action 格式

```json
{
  "action": "reference-files",
  "parameters": {
    "task": "list"
  }
}
```

常見 `task` 包含 `list`、`read`、`search`，實際參數依執行器支援為準。

## 注意事項

- 回覆需標明依據的檔案名稱或可識別來源。
- 不要把沒有讀到的檔案內容說成已確認。
